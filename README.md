# Risto
A project from me using python to create a tauri like expirience with 204mb of storage one exe heh also here's the installation guide.

# Installation Guide
First go here https://www.python.org/downloads/ and download the latest release.
Next go and install a package called pyside6 with this command pip install pyside6
Next install another package called pyinstaller with this command pip install pyinstaller
Next download the latest release zip file and extract it where you want it or just run this in powershell irm https://raw.githubusercontent.com/infinityinquire/Risto/main/install.ps1 | iex.
Next open cmd in the extracted directory and run this command run-risto-test to test it.
If you want to build it run this command run-risto-build
